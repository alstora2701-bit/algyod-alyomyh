# الدليل المحاسبي التفاعلي — مشروع مكتمل (نسخة محلية)

هذا المشروع يحول النسخة الأولية إلى نظام محاسبي تفاعلي محلي "قابل للاستخدام" مع الاحتفاظ بواجهة عربية وRTL. يعمل بالكامل في المتصفح ويغطي:

- شجرة حسابات (Chart of Accounts) جاهزة للتعديل والإضافة.
- تسجيل القيود اليومية متعددة الأسطر مع تحقق أن مجموع المدين = مجموع الدائن.
- دفتر الأستاذ لكل حساب مع رصيد جاري.
- ترحيل قيود تلقائيًا (posting) وحفظ السجلات في localStorage.
- ميزان مراجعة (Trial Balance) وتقرير قائمة الدخل والميزانية العمومية.
- إدارة المخزون (نظام مستمر Perpetual باستخدام FIFO) وحساب تكلفة البضاعة المبوعة تلقائيًا.
- جدول إهلاك للأصول الثابتة (قسط ثابت) مع تحديث تلقائي للأرصدة.
- إمكانية استيراد/تصدير البيانات (JSON) ونسخ احتياطي/استعادة.
- رسومات بيانية توضيحية باستخدام Chart.js.
- واجهة مستخدم RTL مع Tailwind (CDN) وتحسينات تفاعلية.

كيفية التشغيل:
1. قم بتنزيل الملفات الموضحة أدناه في نفس المجلد.
2. افتح `index.html` في المتصفح (يفضل Chrome/Edge).
3. يتم حفظ جميع البيانات محليًا في localStorage للمجال/المسار، ولا توجد أي بيانات تخرج من جهازك.

ملاحظة: هذا تطبيق تعليمي/تجريبي محلي. لتحويله إلى نظام إنتاجي حقيقي ستحتاج إلى:
- قاعدة بيانات من جهة الخادم (Postgres/MySQL/Firestore...).
- تحكم بالوصول والمصادقة.
- سجلات تدقيق (audit trail) ونسخ احتياطية خارجية.
- اختبارات أمنية وأداء.

ملفات مضمنة:
- index.html — الواجهة الأساسية والتحكم.
- app.js — منطق التطبيق (حفظ، قيد، قيادة، تقارير، مخزون، إهلاك).
